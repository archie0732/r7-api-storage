Favorite API

This page provides an API to view the content of favorite.json.

GET Endpoint

Use the following URL to get the JSON data:

https://.github.io//favorite.json

Example Usage

Fetch the JSON data using JavaScript:


fetch('https://.github.io//favorite.json')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error fetching favorite.json:', error));
