# 作業1

試寫一個程式，依據下表規格建立矩形類別(Rectangle)，程式將要求使用者輸入長與寬，以建立矩形物件，並計算該矩形物件的面積與周長，且將結果予以輸出。

## 資料成員

長(length)  `公有(public)`

寬(width) `公有(public)`

面積(area) `私有(private)`

周長(perimeter) `私有(private)`


## 成員函式

設定長寬(setLengthWidth)
公有(public)

計算面積(calArea)
公有(public)

計算周長(calPerimeter)
公有(public)

取得面積(getArea)
公有(public)

取得周長(getPerimeter)
公有(public)


### 範例

- input

```md
請輸入矩形的長度：10 請輸入矩形的寬度：5
```

- output

```md
矩形的長度為：10
矩形的寬度為：5
矩形的面積為：50
矩形的周長為：30
```
